`cmdcomp` generate command shell completion file (`bash` or `zsh`) from config
`json`/`yaml`/`toml` file.

![image](./docs/images/image.png)

## Install

```shell
pip install cmdcomp
```

## Usage

```shell
cmdcomp --file $YOUR_CONFIG_FILE --shell-type bash
```
